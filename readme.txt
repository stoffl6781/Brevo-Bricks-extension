Ein WordPress-Plugin zur Synchronisation von Daten mit Brevo API Schnittstelle.

== Description ==
Brevo Bricks Erweiterung

== Installation ==

1. Lade die Plugin-Dateien in das Verzeichnis `/wp-content/plugins/` hoch oder installiere es direkt über das WordPress-Dashboard.
2. Aktiviere das Plugin über das Menü *Plugins* in WordPress.
3. Navigiere zu *Brevo Bricks Einstellungen*, um API-Einstellungen vorzunehmen.

== Usage ==
Füge die Formular-ID bei den Einstellungen hinzu. 
Im BricksFormular (Editor):
Verwende ein Hiddenfield mit der ID "BrevoListe" für die BrevoListenID. 
Verwende für den Vornamen die FeldID "Vorname".
Verwende für den Nachnamen die FeldID "Nachname".
Verwende für die E-Mail Adresse die FeldID "E-Mail".
